# Card Type Background Images

This folder contains background images for each card type in the auction. Replace or add the following images:

## Background Images Required

### 1. **white.jpg** - Iconic & Bigtime GK Cards
- **Color Scheme:** White/Light
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Clean, elegant white or light background with subtle texture
- **Use Cases:** Iconic (rating 111) and Bigtime GK (rating 108, position GK)

### 2. **black.jpg** - Bigtime Cards
- **Color Scheme:** Black/Dark
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Deep black or very dark background with modern texture
- **Use Cases:** Bigtime (rating 110-108, non-GK positions)

### 3. **blue.jpg** - Epic Cards
- **Color Scheme:** Blue
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Deep blue or royal blue background with dynamic texture
- **Use Cases:** Epic (rating 107-104)

### 4. **gold.jpg** - Highlight Cards
- **Color Scheme:** Gold/Yellow
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Rich gold or warm golden background with elegant texture
- **Use Cases:** Highlight (rating 103-100)

### 5. **silver.jpg** - Silver Cards
- **Color Scheme:** Silver/Gray
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Metallic silver or cool gray background with refined texture
- **Use Cases:** Silver (rating 98-96)

### 6. **bronze.jpg** - Bronze Cards
- **Color Scheme:** Bronze/Brown
- **Dimensions:** 1920x1080px (minimum)
- **Format:** JPG
- **Description:** Rich bronze or warm brown background with texture
- **Use Cases:** Bronze (rating 95-93)

### 7. **default.jpg** (Optional)
- **Color Scheme:** Professional Blue/Dark
- **Description:** Default background for when no player is selected
- **Use Cases:** Waiting state or no player in auction

## How It Works

1. The auction page automatically detects the current player's card type
2. The matching background image loads dynamically
3. A subtle overlay is applied to ensure text readability
4. Smooth fade transition (0.8s) when switching between players of different card types

## File Naming Convention

- All files must be named exactly as listed above
- Use lowercase filenames
- Keep file sizes optimized (< 500KB each recommended)
- Use .jpg format for best performance

## Recommended Sources for Background Images

- **Unsplash** (unsplash.com) - Free high-quality images
- **Pexels** (pexels.com) - Free stock photos
- **Pixabay** (pixabay.com) - Free images
- **Adobe Stock** (stock.adobe.com) - Premium images

## Tips

- Add subtle textures or patterns for visual interest
- Ensure good contrast for text readability
- Use semi-transparent overlays in CSS for text visibility
- Optimize images for web (compress without losing quality)
- Test on different screen sizes
